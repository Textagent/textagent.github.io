# Reusable Action Modal

- Unified `showActionModal()` replaces native `confirm()` and inline rename
- Rename: modal with editable input field, auto-selects filename sans extension
- Duplicate: confirmation modal with blue Duplicate button
- Delete: confirmation modal with red Delete button  
- Enter submits, Escape cancels, click-outside dismisses
- Added input field CSS with blue focus ring and button color variants
